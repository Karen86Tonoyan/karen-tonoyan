## 🚀 Quick Start

```bash
# Termux (Android)
chmod +x install_termux.sh
./install_termux.sh
export OPENAI_API_KEY='your-key'
python openai_integration.py

# Standard Python
pip install -r requirements_alfa_openai.txt
export OPENAI_API_KEY='your-key'
python alfa_openai_core.py
```
